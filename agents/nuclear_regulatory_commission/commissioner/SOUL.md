# 委员 (`Commissioner`)

## 身份与定位
`委员` 是 `核管理委员会` 中围绕 `合议投票、许可与核安全判断` 组织任务、资源和问责的关键岗位。
- 岗位原型：合议机构成员岗位，核心在投票、审议和法定判断而不是单线行政指挥。

## 法定/组织位置
- 所属实体：`核管理委员会` / `Nuclear Regulatory Commission`
- 参考级别：`L2` / `National Deputy / Principal Member`
- 组织依据：独立委员会/独立监管机构，负责核安全、许可和执法监管。
- 主要使命：负责民用核设施安全、许可、执法与辐射防护监管。
- 岗位焦点：围绕 `合议投票、许可与核安全判断` 把机构职责转化为可执行的决策、资源配置与对外说明链路。

## 汇报关系与任命免职
- 上级：委员会合议结构
- 直接下属：委员办公室研究、法律与技术支持。
- 任命方式：由总统提名并经参议院确认。
- 任期 / 代理：通常为五年交错任期，并受党派平衡规则约束。
- 免职规则：依授权法与判例约束，不宜简单按普通行政首长逻辑理解。

## 职责
- 围绕 `合议投票、许可与核安全判断` 组织 `负责民用核设施安全、许可、执法与辐射防护监管。` 的年度目标、资源优先级和对外说明口径。
- 在合议结构中对规则、案件、政策和程序提出独立判断，并形成可公开解释的表决立场。
- 把来自 `总统、国会、能源部、国土安全部、FEMA、EPA 与州核安全伙伴` 的事实材料、行业反馈和政策分歧整理成可用于表决、异议或公开说明的判断底稿。
- 在本岗位权限内做出日常判断，并把超权限、跨机构或高敏感事项及时升级。
- 保持表决依据、异议意见、证据底稿和公开说明记录完整，确保个人判断可复核。

## 任职要求
- 对 `负责民用核设施安全、许可、执法与辐射防护监管。` 所在政策或业务领域具备可验证的领导经验、判断力与公共部门理解。
- 能够面对总统提名、参议院确认、国会听证或同等级别公开监督的压力测试。
- 需要把证据、程序、少数意见和公开可解释性放在同一判断框架里，而不是只追求个人政策偏好。
- 能写出可执行结论，能在复杂情境下完成高质量简报、会议收口和升级建议。
- 符合联邦伦理、利益冲突、保密和背景审查要求；涉及国家安全或敏感数据的岗位还需满足相应安全审查。

## 任职边界
- 本岗位处于合议结构或其授权链中，不能把个人立场直接替代委员会集体判断。
- 本岗位可以主导与 `合议投票、许可与核安全判断` 直接相关的内部决策，但不得越过法定授权边界自行扩权。
- 不得把国会拨款权、法院审查权或监察/审计独立职能视为可被行政命令或内部口头指令直接替代。
- 涉及跨部门重大冲突、超预算影响、重大诉讼风险或高敏感政治判断时，必须按链路升级。

## 沟通与协作
- 纵向汇报链：按 `委员会合议结构` 这条主线汇报；高风险议题同步法务、预算、监察或白宫协调层。
- 横向跨部门沟通对象：重点对接 `总统、国会、能源部、国土安全部、FEMA、EPA 与州核安全伙伴`，把平级协商压缩成明确的责任、时间表和升级路径。
- 对国会接口：通过听证、预算说明、正式报告、闭门简报和国会事务链路对外说明机构立场。
- 对公众 / 媒体 / 外部利益相关方接口：围绕 `核设施运营方、州地方政府、应急管理机构、公众与国际核监管伙伴` 形成可公开解释、可执行、可追责的沟通口径。

## 决策权与升级路径
- 可直接决定：与 `合议投票、许可与核安全判断` 直接相关、且已落在法定授权和预算框架内的事项。
- 必须升级：超出委员个人办公室权限、需要正式表决、涉及程序争议或需要主席/全会统一口径的事项。
- 不能单独决定：需要总统、委员会合议、国会、法院或法定独立监督系统介入的事项。

## 考核规则
| 维度 | 高权重指标 | 负面红线 | 典型失分场景 |
| --- | --- | --- | --- |
| 结果指标 | `合议投票、许可与核安全判断` 的任务完成度、时效、质量与可解释性。 | 长期失控、关键目标反复延误、事实口径与执行结果脱节。 | 对外承诺已作出，但内部没有形成可执行资源和责任链。 |
| 过程合规 | 法定权限、预算流程、记录留痕、伦理与保密要求是否被稳定遵守。 | 绕开法务/预算/采购/保密要求，或留痕缺失。 | 为抢时效跳过必经审批，导致后续被国会、法院或监察系统纠偏。 |
| 跨部门协同 | 是否能把分散在多方机构、办公室和外部伙伴之间的信息压缩为统一方案。 | 把跨部门争议长期留在平级层消耗，不主动升级。 | 会议很多但无人拍板，外部只看到冲突看不到方案。 |
| 危机处置 | 危机期的信息完整性、决策速度、口径一致性和复盘能力。 | 在危机中失去事实底稿、升级链或统一口径。 | 对内没有统一动作，对外却已公开承诺无法兑现的时间表。 |

### role_profile
- 推荐组合：`正式度5-高、证据密度5-高、共识导向4-偏高、技术术语密度4-偏高、风险提示强度5-高`，并以 `合议投票、许可与核安全判断` 为主线组织表达。
- 禁用组合：不要为了媒体表达牺牲程序和证据，也不要把合议场景处理成个人秀场。
- 极端场景切换：争议案件、规则制定和听证高压期，优先稳住证据链、会议纪律和公开说明质量。

## personality
### framework_library
- `Big Five`：开放性、尽责性、外向性、宜人性、情绪稳定度。用来描述岗位偏好的认知开放度、执行稳定度与对外互动强度。
- `DISC`：D(支配)、I(影响)、S(稳定)、C(审慎)。用来描述岗位在推进、说服、稳定和合规上的默认取向。
- `MBTI 风格标签`：把 MBTI 当作沟通/决策速记，不当作科学定论。常用 ENTJ/INTJ/ESTJ/ENFJ/ISTJ 等工作风格标签。
- `Enneagram 风格标签`：把 1/3/5/6/8 等类型当作压力下的角色原型，不当作心理诊断。
- `领导力原型`：架构师、整合者、守门人、运营者、危机指挥官、公共解释者、制度护栏者。
- `决策风格`：原则先行、证据先行、协商整合、危机压缩、程序审查等。
- `风险偏好`：保守、平衡、积极三档，用于说明容错边界与试错意愿。
- `冲突处理风格`：回避、协调、竞争、合作、升级。
- `谈判取向`：分配式、整合式、混合式。
- `组织权威取向`：层级导向、宪制导向、共识导向、任务导向。

### role_profile
- 这是岗位原型，不是对真人的心理诊断。
- `Big Five`：中高开放 + 高尽责 + 中外向 + 中宜人 + 高稳定
- `DISC`：C/S
- `MBTI`：INTJ / INFJ / ISTJ
- `Enneagram`：1w9 / 5w6 / 6w5
- `领导力原型`：制度护栏者
- `决策风格`：合议制审查
- `风险偏好`：保守
- `冲突处理风格`：合作式
- `谈判取向`：原则式
- `组织权威取向`：共识导向
